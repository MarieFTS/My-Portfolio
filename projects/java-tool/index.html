Overview

This educational project demonstrates core Java competencies through a feature-rich CLI data analysis tool. Without relying on external libraries, the application handles CSV parsing, statistical computation, data manipulation, and visualization. The architecture showcases OOP principles including encapsulation, composition, and stream processing—all compiled to a single JAR file that works with any CSV dataset.

Process & Methodology

1

CSV Parser

Built BufferedReader-based parser mapping column headers to row values with robust error handling.

2

Row Model

Created inner class Row with get(column) and getDouble(column) helper methods for type safety.

3

Descriptive Stats

Computed count, mean, std dev, min, Q1, median, Q3, max using sorted collections.

4

Frequency Analysis

Used Java Streams groupingBy() + counting() to rank categorical values by frequency.

5

Filter & Sort

Implemented >, >=, <, <=, =, contains operators with numeric/string dispatch logic.

6

Aggregate & Visualize

GroupBy with SUM/AVG/COUNT/MAX/MIN; ASCII bar charts scaled to terminal width.

// Descriptive statistics
public void describe(String column) {
    List nums = rows.stream()
        .map(r -> r.getDouble(column))
        .filter(v -> !Double.isNaN(v))
        .sorted()
        .collect(Collectors.toList());

    double mean   = nums.stream()
                        .mapToDouble(Double::doubleValue)
                        .average().orElse(0);
    double median = percentile(nums, 50);
    double stddev = stdDev(nums, mean);

    System.out.printf("Mean: %.2f | Median: %.2f | StdDev: %.2f%n",
                      mean, median, stddev);
}
