Fraud Detection Dashboard

Designed and implemented an interactive fraud detection dashboard integrating multi-source transaction data. Created visualizations to monitor patterns, detect anomalies, and flag suspicious transactions in real time.

Sep 2023 – Nov 2023 Individual Project Tableau + MySQL Real-time Dashboard

Overview

This project combined SQL data engineering with Tableau visualization to create a real-time fraud detection system. By analyzing 1.2M+ transactions, we developed automated anomaly detection queries and interactive dashboards that enable rapid identification and response to suspicious activity across multiple dimensions: time, geography, category, and customer behavior patterns.

Process & Methodology

1

Data Ingestion

Loaded raw transaction CSV into MySQL, validated data quality, and resolved schema mismatches.

2

Feature Engineering

Calculated distance between customer and merchant using Haversine formula, created time and amount buckets.

3

Anomaly Queries

Identified transactions exceeding 3 standard deviations from mean; flagged cards with 10+ daily transactions.

4

Tableau Data Source

Created export views: daily summary, geo summary, and full enriched transaction table.

5

Dashboard Build

Built 5 Tableau sheets: time series trends, category heatmap, geo map, KPI cards, and anomaly table.

6

Interactivity

Added filters for date range, state, category, and fraud flag to enable deep-dive analysis.

High-Risk Transaction Detection

SQL 
-- High-risk transaction detection
SELECT transaction_id, trans_datetime,
       merchant, category, amt,
       distance_km, is_fraud
FROM transactions_enriched
WHERE amt > (
    SELECT AVG(amt) + 3 * STDDEV(amt)
    FROM transactions
)
ORDER BY amt DESC;
