COVID-19 Data Analysis

Analyzed global COVID-19 trends including case counts, death rates, vaccination coverage, and regional patterns using Python data science libraries. Dataset from Our World in Data (OWID).

Academic Project Individual Project Python 4 Visualizations

Overview

This comprehensive data analysis project examined global COVID-19 trends across 200+ countries using Python data science tools. Through careful data cleaning, feature engineering, and visualization, we discovered meaningful patterns in case fatality rates, vaccination effectiveness, and regional pandemic response. The analysis revealed a strong negative correlation between vaccination rates and case fatality rates, validating vaccine efficacy across diverse populations.

Process & Methodology

1

Data Loading

Loaded OWID COVID CSV with 200+ countries and 60+ columns using Pandas read_csv().

2

Data Cleaning

Removed aggregate rows (continents/World), filled missing values, selected 15 key columns.

3

Feature Engineering

Computed case fatality rate, cases per million, vaccination rate, and 7-day rolling averages.

4

Global Summary

Printed aggregate statistics: total cases, deaths, vaccinated count, and overall CFR.

5

Visualizations

Generated 4 charts: Top 10 countries, global trend, vaccination vs CFR scatter, continent heatmap.

6

Insights & Reporting

Found negative correlation between vaccination rate and case fatality rate; documented findings.

df['vax_rate'] = np.where(
    df['population'] > 0,
    (df['people_fully_vaccinated'] / df['population']) * 100, 0
)
df['7day_avg_cases'] = (
    df.groupby('location')['new_cases']
      .transform(lambda x: x.rolling(7, min_periods=1).mean())
)
